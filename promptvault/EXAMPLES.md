# 📝 Example Prompts

Here are some starter prompts to get you going with PromptVault!

## 📧 Business & Communication

### Professional Email Writer
```
Write a professional email to {recipient} about {topic}.

Tone: {tone}
Length: {length}
Action required: {action}
```
**Tags**: `business`, `email`, `communication`

### Meeting Summary
```
Summarize the following meeting notes:

{meeting_notes}

Format the summary with:
1. Key decisions made
2. Action items (with owners)
3. Next steps and deadlines
```
**Tags**: `productivity`, `meetings`, `business`

### LinkedIn Post Creator
```
Create an engaging LinkedIn post about {topic}.

Tone: {professional/casual/thought-leadership}
Include: Personal story, key insights, call-to-action
Length: 150-200 words
```
**Tags**: `social-media`, `marketing`, `linkedin`

## 🎯 Tips for Creating Great Prompts

1. **Be Specific**: The more details you provide, the better the output
2. **Use Variables**: Add `{variables}` for reusable prompts
3. **Structure Matters**: Use numbered lists, bullets, and sections
4. **Set Context**: Define tone, audience, and purpose
5. **Examples Help**: Include "Format like this:" with examples
6. **Iterate**: Refine prompts based on results
7. **Tag Wisely**: Use consistent tags for easy finding
8. **Test First**: Try prompts on different AI models
