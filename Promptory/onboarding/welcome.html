Google Sign-In Setup (Free)

To enable Google sign-in, you need to configure Supabase:

1. Go to Supabase Dashboard → Authentication → Providers → Google
2. Create a project in Google Cloud Console
3. Enable OAuth consent screen (External, Testing mode = free)
4. Create OAuth 2.0 Client ID (Web application)
5. Add Authorized redirect URI: https://vofgfvlgchqheksvlibl.supabase.co/auth/v1/callback
