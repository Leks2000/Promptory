📝 PromptVault

AI Prompt Manager — Save, organize, and instantly insert prompts into ChatGPT, Claude, Gemini, and more.

Note: This is a preview of the extension popup. To use the full functionality, install the extension in Chrome via chrome://extensions → Developer mode → Load unpacked.

✨ Key Features

  • 📁 Organize prompts in folders with custom icons
  • 🔥 Quick access to favorite prompts
  • 🌐 Explore community prompt library
  • ⚡ 4 customizable hotkeys for instant insert
  • 🔄 Cloud sync with Supabase
  • 🎨 Beautiful dark/light themes

🎯 Smart Variables

Use {variable} syntax in prompts. When inserting, a dialog appears to fill in the values.

🤖 Platform Detection

Automatically detects ChatGPT, Claude, Gemini, Perplexity, Poe, and more. Inserts directly into the chat input.

⌨️ Keyboard First NEW

Alt+S to open. Assign up to 4 prompts to custom hotkeys for instant insertion.

📊 Usage Tracking

See how often each prompt is used. Optimize your workflow based on actual usage data.
